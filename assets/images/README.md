# Images Folder Structure

Store project-managed images here to keep a clean handoff-friendly structure.

## Folders

- `logo/` - brand assets (`logo.svg`, alternates)
- `hero/` - homepage hero images/video poster frames
- `suites/` - room images
- `gallery/` - gallery images
- `sections/` - about, amenities, and other section-level visuals

## Naming

- Use lowercase and hyphens: `hero-main-01.jpg`
- Keep original source files outside git if very large.
- Prefer optimized web formats for production (`webp`, `jpg`, `png`, `svg`).
