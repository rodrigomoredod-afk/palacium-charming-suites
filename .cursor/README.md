# Cursor Workspace Conventions

This folder stores execution-oriented assets that agents can run directly.

## Folders

- `.cursor/plans/` - accepted plans and active implementation checklists.
- `.cursor/prompts/` - copy-paste prompts for recurring tasks.

## Workflow

1. Draft or update strategy docs in `docs/`.
2. Convert actionable items into `.cursor/plans/` and `.cursor/prompts/`.
3. Execute tasks.
4. Reflect final outcomes back into `docs/`.
